# OOPS 2026: Object-Oriented Programming Notes

## Why This Repo Exists
This repository is designed to provide a clear, concise, and visually appealing set of notes and examples for Object-Oriented Programming (OOP) concepts. It aims to help learners and programmers quickly understand and revise the core principles of OOP, especially in C++.

## What You Will Learn
- The four pillars of OOP: Encapsulation, Inheritance, Polymorphism, and Abstraction
- Constructors, destructors, and memory management
- Advanced topics like the Diamond Problem, object slicing, virtual destructors, VTABLE/VPTR, and friend functions
- Practical code examples and real-world analogies
- Interview tips and common pitfalls

## Repository Structure
- Each major OOP topic is covered in a dedicated, well-formatted markdown file.
- Example code snippets are provided for clarity.
- Files are named and ordered for easy navigation.

**File-by-file breakdown:**

- **01_Introduction_To_OOP.md**: Introduction to OOP concepts, classes, objects, and real-world analogies.
- **02_oops.cpp**: Basic C++ code demonstrating class, properties, methods, and encapsulation.
- **03_Four_Pillars_Of_OOP.md**: Explanation of the four pillars of OOP: Encapsulation, Inheritance, Polymorphism, and Abstraction.
- **04_Encapsulation.cpp**: C++ code example illustrating encapsulation, getters, and setters.
- **05_Constructors_and_Destructors.md**: Notes on constructors, destructors, memory management, shallow/deep copy, and the `this` pointer.
- **06_Constructor.cpp**: C++ code showing different types of constructors and constructor overloading.
- **07_Shallow_And_Deep_Copy.cpp**: C++ code demonstrating shallow and deep copy with dynamic memory.
- **08_Destructor.cpp**: C++ code example of destructors and memory cleanup.
- **09_Inheritance.cpp**: C++ code showing single, multi-level, multiple, hierarchical, and hybrid inheritance.
- **10_MultiLevel_Inheritance.cpp**: C++ code focused on multi-level inheritance.
- **11_Multiple_Inheritance.cpp**: C++ code focused on multiple inheritance.
- **12_Hierarchical_Inheritance.cpp**: C++ code focused on hierarchical inheritance.
- **13_Polymorphism.cpp**: C++ code and comments explaining compile-time and runtime polymorphism.
- **14_Compile_Time_Polymorphism.cpp**: C++ code for function and operator overloading (compile-time polymorphism).
- **15_RunTime_Polymorphism.cpp**: C++ code for function overriding and virtual functions (runtime polymorphism).
- **16_Abstraction.cpp**: C++ code and comments on abstraction, abstract classes, and pure virtual functions.
- **17_Static_Keyword.cpp**: C++ code and notes on static variables, static objects, and their behavior.
- **18_Diamond_Problem.md**: Explanation and code for the diamond problem in multiple inheritance and its solution with virtual inheritance.
- **19_Hot_Topics.md**: Advanced OOP topics: object slicing, virtual destructors, VTABLE/VPTR, friend functions, and interview tips.
